Internet Voting System (Final Year Project)

A blockchain-based voting system developed for my final-year dissertation, focused on secure, transparent, and verifiable internet elections using a permissioned peer-to-peer network.

I implemented a Proof-of-Authority consensus mechanism where a designated leader proposed blocks, and validator nodes confirmed transactions before finalization. The system emphasizes vote integrity, end-to-end verifiability, and resistance to tampering.

Key features included:

  • Custom-built peer-to-peer networking layer using Python sockets.
  • Validator coordination for consensus with block-level quorum checks (70%+).
  • Digital signature-based vote authentication using RSA encryption.
  • Integrated voter eligibility verification and anonymous ballot handling.
  • Optional phone app for voter identity verification using NFC-enabled ID cards.

This project deepened my knowledge of cryptographic protocols, distributed systems, and the challenges of secure digital voting. It reinforced my interest in backend and security-focused engineering, particularly in the context of real-world system constraints.
